🗳️ Neta Promise Tracker

Track Political Promises in India

A modern, responsive web application to track political manifesto promises and verify their fulfillment status with authentic sources.

✨ Features

📊 Track Promises - Filter by state, district, and election year
✅ Verify Status - Completed, Partially Completed, or Not Completed
🔗 Source-Based Proof - Every promise backed by authentic sources
🔍 Search Functionality - Find promises by keyword
📱 Responsive Design - Works on all devices
🚀 Offline Ready - No internet required after loading


📁 Files
neta-promise-tracker/
├── index.html    # All 4 pages (Home, About, Tracker, Contact)
├── styles.css    # Complete styling
├── script.js     # Functionality + data
└── README.md     # This file
Just 3 files! No dependencies, no build tools.

🚀 Quick Start

Download all 3 files
Keep them in the same folder
Open index.html in your browser
Start tracking promises!
